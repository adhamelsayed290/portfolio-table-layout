# HTML Portfolio Assignment

##  Live Demo
 https://portfolio-table-layout.vercel.app/

---

##  Project Overview
This project is a simple portfolio webpage built using pure HTML as part of a frontend development assignment.  
It focuses on practicing table-based layouts and HTML form elements to build a structured static webpage.

---

##  Technologies Used
- HTML5

---

##  Features
- Table-based layout structure
- Personal information section with profile image and social links
- Skills section
- Experience timeline using HTML tables
- Contact form with multiple input types (text, email, radio buttons, checkboxes, dropdowns)

---

##  Learning Goals
This project was built to strengthen core HTML fundamentals, including:
- Structuring pages using tables
- Creating and handling forms
- Understanding semantic HTML structure
- Building complete static web pages
